Security Audits
===============

This is the canonical topic page for security audits.

Scope
-----

Use this topic for verified security risks, privacy exposure, credential
handling issues, and deployment-surface tradeoffs. Keep accepted risks
explicitly labeled as such instead of mixing them with ordinary bugs.

Row schema
----------

- ``Priority``
- ``Audience`` (``user``, ``deployer``, ``maintainer``)
- ``Code target``
- ``Verified risk``
- ``Proposed mitigation``
- ``Status``
